# MA-8 — Engagement-Fit

**Status:** DRAFT (master plan §6.4; JR review-iterate before manifest freeze)

## What this rubric scores

The findings + proposal align with the `capability_registry`. Tier-mapping serves a pitch for a $15K+ engagement (not a $1K-only artifact). T+60d engagement-conversion signal corroborates this rubric over time — but at audit-render time, the rubric judges whether the deliverable IS pitching a credible engagement.

## What "good" looks like

- Top severity findings map to `build_it` or `run_it` engagements (where the agency revenue lives)
- `fix_it` tier entries cover real one-off scope, not table-stakes
- `build_it` tier entries reference the agency's actual production capability (per registry)
- Proposal narrative anchor names the strategic shape ("the highest-leverage work is rebuilding positioning + standing up a CRO experimentation program")
- An ICP buyer reading the proposal can imagine engaging the agency for the named scope

## What "bad" looks like

- Top findings all map to `fix_it` (no $15K+ pull)
- Proposal entries are generic SaaS-vendor-style ("we'll help you grow")
- Capability mapping is mismatched (recommending an engagement the registry doesn't list)
- No narrative anchor; proposal reads as a price list
- Proposal pitches engagements the prospect has no shape-fit for (proposing run-it-content-ops to a 3-person dev-tools startup that needs a positioning rebuild)

## Score scale (0-10)

- **0-2** Findings + proposal don't connect to engagement; reads as deliverable-only
- **3-4** Some build_it/run_it entries; proposal is generic
- **5-6** Most top findings map to build_it/run_it; proposal references registry; narrative anchor weak
- **7-8** Proposal is shape-fit for the prospect; tier laddering is credible; narrative anchor names strategic shape
- **9-10** Above + the proposal feels purpose-built for the prospect's specific situation; T+60d conversion likelihood is high

## Anchors for severity-of-MA-8-failure

- Top severity-3 findings all map to fix_it: severity-of-failure 3
- Proposal narrative anchor missing or generic: severity-of-failure 2
- Capability mapping mismatched (recommending non-registry capabilities): severity-of-failure 3
- Proposal pitches engagements the prospect has no shape-fit for: severity-of-failure 2

# ad_static / value-prop archetype

Ad creative anchored on the value-prop headline. Visual anchors the
prop; text-overlay <20% pixel area.

## Prompt template

A clean static ad creative. Subject: {subject}, illustrating the
value proposition without competing for attention with the
forthcoming text overlay. Lighting: {lighting}. Composition: clear
focal subject; ample whitespace in the upper third where the value-
prop headline (Pillow-composited) will land. Palette:
{brand_palette_hex}. Aspect ratio per platform — declare in
meta.json. NO rendered text. NO call-to-action text in the image
(Pillow-composited after).

## Slots

- `{subject}`           — visual that anchors the value prop
- `{lighting}`          — lighting register
- `{brand_palette_hex}` — brand HEX codes
- platform-specific aspect ratio set in meta.json

## When to use

- Meta/LinkedIn paid ads where headline IS the value prop
- Awareness-stage ad creative
